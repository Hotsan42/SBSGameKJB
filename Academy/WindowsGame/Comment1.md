## MarkDown

### markdown이란?
- 프로그래머가 간단하게 문서를 만들 때 사용하는 확장자 파일
- 간단한 표현만으로도 스타일링을 쉽게 줄 수 있다.

### markdown 문법

1. 줄변경 : ---
---
2. 표 나타내기
```
|Backend|lasd|123|123|123123|

```
3. 제목표현
```
#제목1
##제목2
###제목3
####제목4
#이 많아질수록 글자크기가 작아진다
```
4. 볼드체/ 취소표현 / 이텔릭표현
```
*이텔릭체*
**볼드체**
~~취소표현~~
```
5. 목록
```
- 첫번째
	-두번째
	- 세번째
```
6. 하이퍼링크
```
[GOOGLE](https://google.com)
```
7. 이미지
```
![대체텍스트](이미지링크)
```
8. 인용문
>인용문1
>>인용문2
>>>인용문3
